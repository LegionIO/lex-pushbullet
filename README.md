# lex-pushbullet

Pushbullet push notification integration for [LegionIO](https://github.com/LegionIO/LegionIO). Send notifications to Pushbullet-connected devices from within task chains.

## Installation

```bash
gem install lex-pushbullet
```

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
- Pushbullet account
